  • What is the difference between var and let?
    • Var is function scope whereas let is block scope, meaning that a var declared within a block (such as a for loop) is accessible outside the loop while an object declared/defined with let is only accessible within the block.
    • An object declared with var can be redeclared whereas an object declared with let cannot be redeclared.
    • An object declared as var can be hoisted.
  • What is the difference between var and const?
    • Var has function scope whereas const has block scope.
    • Var can be redeclared as well as reassigned whereas const can only be declared once and cannot be reassigned. For example, if a const is declared to be an array it will always be an array although the contents of that array can be changed. Conversely, a var may be declared to be an array then redefined to be any other object such as Number or String.
    • Var can be hoisted wherease const cannot.
  • What is the difference between let and const?
    • Let and const cannot be redeclared but let can be reassigned whereas const cannot.
  • What is hoisting?
    • Hoisted variables can be accessed before being declared within the source code.
